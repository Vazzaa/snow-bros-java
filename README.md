# p-comision-11
Proyecto:
Modelado e implementación de una versión simplificada de un juego
-el proyecto consiste en el diseño e implementación en Java de una versión simplificada e inspirada pero modificada de Snow Bros
Este modelado tiene:
-2 dominios de aplicacion donde solamente cambian las entidades
-3 modos de juego (Clasico,Contra reloj,Supervivencia)
-2 niveles (en este caso dos pisos)
-5 Enemigos
-5 PowerUps
-2 Jefes (uno en cada nivel)
