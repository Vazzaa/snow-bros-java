# Snow Bros: Java Edition (Comisión 11)

Este proyecto es una implementación en **Java SE** de una versión simplificada y modificada del clásico arcade *Snow Bros*. Fue desarrollado como el proyecto final para la materia **Taller de Programación** (Ciencias de la Computación, UNS).

## Resumen del Proyecto
El juego mantiene la esencia del original pero introduce mecánicas personalizadas y una arquitectura flexible que permite alternar entre dos dominios de aplicación (donde cambian las entidades manteniendo la lógica de juego).

### Características Principales:
* **Modos de Juego:** * 🏆 **Clásico:** La experiencia tradicional.
    * ⏱️ **Contra reloj:** Desafío de velocidad.
    * 🛡️ **Supervivencia:** Aguantar oleadas el mayor tiempo posible.
* **Estructura de Niveles:** 2 niveles (pisos) completamente funcionales.
* **Contenido:**
    * 5 tipos de Enemigos con comportamientos únicos.
    * 5 PowerUps para potenciar las habilidades del jugador.
    * 2 Jefes finales (uno por cada nivel) con patrones de ataque específicos.

## Aspectos Técnicos
* **Lenguaje:** Java.
* **Interfaz:** Swing / AWT para el renderizado de gráficos y manejo de eventos.
* **Arquitectura:** Implementación de patrones de diseño (como **Observer**) para desacoplar la lógica del dominio de la visualización, facilitando la escalabilidad de los dos dominios de aplicación.
